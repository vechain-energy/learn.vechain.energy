# VET/USD

Provides the current USD price for Vechains native token VET.

## Datasheet

* Address: `0x49eC7192BF804Abc289645ca86F1eD01a6C17713` @ `MainNet`
* Feed ID: `vet-usd` / `0x7665742d75736400000000000000000000000000000000000000000000000000`
* Heartbeat: `24 Hours`
* Max. Deviation: `1%`
* Update Interval: `5 Minutes`
* Status URL: https://oracle.vechain.energy/vet-usd
* CCIP Resolver: https://oracle.vechain.energy/vet-usd/resolver

## Sources

* [Coinbase](https://www.coinbase.com)
* [KuCoin](https://www.kucoin.com)
* [coincap](https://coincap.io)
* [CoinMarketCap](https://coinmarketcap.com)
* [Chainlink VET / USD](https://data.chain.link/bsc/mainnet/crypto-usd/vet-usd) using RPC ankr.com and mewapi.io
