# VTHO/USD

Provides the current USD price for VechainThor, the native gas token for Vechain.

## Datasheet

* Address: `0x49eC7192BF804Abc289645ca86F1eD01a6C17713` @ `MainNet`
* Feed ID: `vtho-usd` / `0x7674686f2d757364000000000000000000000000000000000000000000000000`
* Heartbeat: `24 Hours`
* Max. Deviation: `1%`
* Update Interval: `5 Minutes`
* Status URL: https://oracle.vechain.energy/vtho-usd
* CCIP Resolver: https://oracle.vechain.energy/vtho-usd/resolver

## Sources

* [Coinbase](https://www.coinbase.com)
