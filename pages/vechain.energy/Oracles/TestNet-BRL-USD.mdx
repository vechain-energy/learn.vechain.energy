# BRL/USD

Provides the current USD price for BRL to support Brazilian BRL conversion for USD prices.

<small>
_Note: The data is sourced from Chainlinks BSC & Base Oracle: https://data.chain.link/feeds/bsc/mainnet/brl-usd
</small>

## Datasheet

* Address: `0xdcCAaBd81B38e0dEEf4c202bC7F1261A4D9192C6` @ `TestNet`
* Feed ID: `brl-usd` / `0x657572742d757364000000000000000000000000000000000000000000000000`
* Heartbeat: `24 Hours`
* Max. Deviation: `1%`
* Update Interval: `5 Minutes`
* Status URL: https://oracle.vechain.energy/brl-usd
* CCIP Resolver: https://oracle.vechain.energy/brl-usd/resolver

## Sources

* [Chainlink BRL / USD on Base](https://data.chain.link/feeds/base/base/brl-usd) using RPC ankr.com
* [Chainlink BRL / USD on BSC](https://data.chain.link/feeds/bsc/mainnet/brl-usd) using RPC bscrpc.com
