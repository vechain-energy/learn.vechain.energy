## Link

The link the Swap Provider is shown in the list and linked in the details.

## Colors

The following options are available:

| Option | Comment |
| :----- | :------ |
| Primary | Serves as seed to calculate the basic colors (Buttons, Loading Indicators, etc.) |
| Text | Color of Text & QR Codes. Other text colors are derived from it. |
| Link | Color of Text-Links. |

## Theme

- **Dark Mode** uses a dark color palette.
- **Light Mode** uses a light color palette. (Standard)

## Clone

Cloning a Swap Provider copies the configuration and creates a disabled version.


## Enable / Disable

Disabling a Swap Provider pauses access to its website. It can be resumed at any later state.
