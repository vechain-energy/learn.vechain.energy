# Options

<Callout type="info">
Changes to a Theme require up to 15 minutes to take effect.
</Callout>


## Colors

The following options are available:

| Option | Comment |
| :----- | :------ |
| Primary | Serves as seed to calculate the basic colors (Buttons for example) |
| Text | Regular Text, secondary texts are calculated based on the configured color |
| Link | Links, hover effects are calculated based on the configured color |

## Theme

- **Dark Mode** uses a dark color palette.
- **Light Mode** uses a light color palette. (Standard)

## Clone

Cloning a Swap Provider copies the configuration and creates a disabled version.


## Enable / Disable

Disabling a Swap Provider pauses access to its website. It can be resumed at any later state.
