# Auth0

Enterprise Connections support OpenID Connect with these settings:

| Configuration | Value                                                            |
| ------------- | ---------------------------------------------------------------- |
| Issuer URL    | https://auth.api.vechain.energy/.well-known/openid-configuration |
| Type          | Back Channel                                                     |
| Client ID     | none                                                             |
| Client Secret | none                                                             |
| Scopes        | openid profile                                                   |
